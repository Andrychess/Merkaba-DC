# Merkaba — первый вход

Merkaba — личный блокнот с синхронизацией через **ваш** Яндекс.Диск. Каждый пользователь работает со своим аккаунтом: заметки хранятся в папке `Merkaba/` на вашем Диске.

---

## Что понадобится

1. **Аккаунт Яндекса** с подключённым Яндекс.Диском.
2. **OAuth-приложение** на стороне Яндекса (Client ID и Client secret) — один раз на установку, если разработчик не вложил ключи в сборку.
3. **Установленное приложение Merkaba** (Windows или Linux).

---

## Вариант А: вам передали готовую сборку

Если вы получили Merkaba от знакомого или скачали готовый установщик **с уже настроенными ключами**, достаточно:

1. Установить и запустить Merkaba.
2. Нажать **«Войти через Яндекс»**.
3. Ввести код на странице [oauth.yandex.ru/device](https://oauth.yandex.ru/device) (или [ya.ru/device](https://ya.ru/device)).
4. Подтвердить доступ к Диску в браузере.

После входа приложение создаст на вашем Диске папку `Merkaba/` и загрузит заметки локально.

---

## Вариант Б: нужно настроить OAuth самостоятельно

Выполните шаги ниже **один раз**. Ключи сохранятся на вашем компьютере.

### Шаг 1. Создайте OAuth-приложение

1. Откройте [oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new).
2. Заполните форму:
   - **Название** — любое, например `Merkaba`.
   - **Платформы** — выберите **«Веб-сервисы»** (или аналог для веб-приложения).
   - **Redirect URI** — укажите ровно:
     ```
     https://oauth.yandex.ru/verification_code
     ```
   - **Права доступа** — включите доступ к **Яндекс.Диску** (чтение и запись файлов).
3. Сохраните приложение.
4. Скопируйте **ClientID** и **Client secret** (секрет показывается один раз — сохраните его).

### Шаг 2. Введите ключи в Merkaba

1. На экране входа нажмите **«Вставить новый ClientID»**.
2. Вставьте **ClientID** и **Client secret**.
3. Нажмите **«Сохранить и проверить»**.

Если Яндекс не находит приложение, проверьте Redirect URI и что приложение сохранено без ошибок.

### Шаг 3. Войдите в свой аккаунт

**Способ 1 — рекомендуется (код на устройстве):**

1. Нажмите **«Войти через Яндекс»**.
2. Скопируйте показанный код.
3. Откройте [oauth.yandex.ru/device](https://oauth.yandex.ru/device) и введите код.
4. Войдите в **свой** аккаунт Яндекса и разрешите доступ.

**Способ 2 — OAuth-токен вручную:**

1. На [oauth.yandex.ru](https://oauth.yandex.ru) выберите своё приложение и нажмите **«Получить OAuth-токен»**.
2. В Merkaba нажмите **«У меня есть OAuth-токен»**, вставьте токен и войдите.

> Токен, полученный вручную, может иметь ограниченный срок жизни. Device flow обычно удобнее для постоянной работы.

## Где хранятся заметки

| Место | Путь |
|-------|------|
| Облако (ваш Диск) | `Merkaba/` в корне Яндекс.Диска |
| Локальная копия на ПК | папка `vault` в данных приложения (см. Настройки → «Локальный кэш») |

Заметки — обычные файлы `.md` в подпапках `notes`, `projects`, `daily` и в созданных вами пространствах.

---

## Смена аккаунта на этом компьютере

1. Откройте **Настройки** (иконка шестерёнки).
2. Нажмите **«Выйти»**.
3. Войдите под другим Яндекс-аккаунтом.

**Важно:** при смене аккаунта на одном профиле Windows локальный кэш может содержать файлы предыдущего пользователя. Чтобы не смешать данные, после выхода удалите папку локального кэша (путь указан в Настройках) или попросите разработчика включить автоматическую очистку при выходе.

На разных учётных записях Windows у каждого пользователя свой кэш — дополнительных действий не требуется.

## Частые вопросы

### Могут ли двое пользоваться одной установкой по очереди?

Да, если каждый входит под **своим** Яндексом. Данные в облаке у каждого свои. Не передавайте друг другу один OAuth-токен — у каждого должен быть свой вход.

### Нужен ли интернет?

Для первого входа и синхронизации — да. После загрузки заметок можно редактировать локально; изменения отправятся на Диск при следующей синхронизации (кнопка в шапке или автоматически).

### Ошибка 423 / «файл заблокирован»

Яндекс.Диск временно блокирует частые записи. Подождите несколько секунд и нажмите **«Синхронизировать»** в шапке.

### Где посмотреть заметки без Merkaba?

Откройте [disk.yandex.ru](https://disk.yandex.ru) → папка **Merkaba**. Файлы можно читать и скачивать в браузере.

## Краткая шпаргалка

```
1. OAuth-приложение на oauth.yandex.ru (если ключей нет в сборке)
2. Client ID + Secret → Merkaba → «Вставить новый ClientID»
3. «Войти через Яндекс» → код на oauth.yandex.ru/device
4. Готово: заметки в Merkaba/ на вашем Диске
```
